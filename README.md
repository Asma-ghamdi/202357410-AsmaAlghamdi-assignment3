# Personal Portfolio Website

This project is a simple personal portfolio website created as part of Assignment 1.  
It is meant to introduce basic web development concepts using HTML, CSS, and JavaScript.

The website includes a short introduction about me, a projects section, and a contact section.  
It also includes a small JavaScript feature that displays a greeting based on the time of day.

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## How to Run the Project
1. Download or clone the repository.
2. Open the `index.html` file in any web browser.
3. The website will load locally without the need for any additional setup.

---

## Features
- **About Me section** introducing background and goals
- **Projects section** with images and expandable project descriptions
- **Contact form** with validation and feedback messages
- **Navigation menu** for quickly moving between sections
- **Responsive layout** that adapts to different screen sizes
- **Greeting message** that changes depending on the current time of day
- **Theme toggle button** allowing users to switch between light and dark mode
- **Project detail buttons** that show or hide additional project information
